🍿 WatchWhileEating

A front-end for !mealtimevideos@lemmy.cafe. I built it because I kept digging through that community while eating and wanted something that felt more like browsing than scrolling a forum thread.

It's a single HTML file — vanilla JS, the Lemmy API, YouTube embed. No build step, no framework, no tracking.

The code is open source. If something breaks or you have an idea, find me on Lemmy.
